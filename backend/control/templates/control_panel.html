Usuários

1. Para acessar o controle de usuários use o botão "Usuários" no cabeçalho do site.

2. Na página de controle de usuários você vai encontrar um formulário para adicionar novos usuários e abaixo dele uma lista dos usuários já inseridos no sistema.

3. Na mesma lista existem botões de ações sendo elas "Editar" e "Deletar".

4. O botão "Editar" leva para outra página onde as informações podem ser alteradas.

5. O botão "Deletar" claramente apaga completamente as informações. Use com cuidado!

6. Staff significa "Admininistrador", ou seja, o usuário com maior controle do sistema.

7. Status é o estado do acesso do usuário, se o usuário estiver ativo ele pode acessar o sistema e se estiver desativado ele não pode acessar.
