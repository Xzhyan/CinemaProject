## Sobre

O Painel de Controle foi criado usando Django, templates e, claro, Django REST Framework. Ou seja, ele faz o gerenciamento do banco de dados e envia as informações por API para o site principal, que foi desenvolvido em React. O site principal recebe, organiza e exibe as informações de forma dinâmica, garantindo uma melhor experiência para o cliente.

---

## Funcionalidades

O Painel de Controle possui as seguintes telas para gerenciamento:

1. Usuários
2. Tipos de Categorias
3. Categorias
4. Gêneros
5. Filmes
6. Sessões
7. Promoções

> Em todas as telas é possível listar, criar, editar e deletar informações.

---

## Tecnologias Utilizadas

O sistema foi desenvolvido usando Python, especificamente o framework Django. As telas foram criadas utilizando templates HTML e Tailwind CSS para estilização, e o banco de dados utilizado foi o SQLite.

[Python](https://www.python.org/)
[Django](https://www.djangoproject.com/)
[Django REST Framework](https://www.django-rest-framework.org/)
[Tailwind CSS](https://tailwindcss.com/)
[SQLite](https://sqlite.org/)
